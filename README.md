# DNet
# --- For Mac Only ---
# What to do?
- Click on DNet.zip
- Click download
- Download onto Desktop folder (It will only work there)

# Once downloaded
- Open DNet
- Click "Setup"
- Click "Start"
- Close all terminals once opened after process is completed

# If "Setup" not working
- Right click - or two finger click - on "Setup"
- Click "Open"
